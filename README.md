## What AzireVPN Is

AzireVPN is a desktop VPN client for Windows users who want a simpler way to control encrypted traffic. It pairs a minimal GUI with privacy-oriented controls, making it useful for routine browsing, public Wi-Fi, and general connection handling.

The application works with both WireGuard and OpenVPN, giving users flexibility across different preferences and network conditions. With options like a kill switch, multi-hop routing, and ad and tracker blocking, it provides additional control over how traffic is routed and filtered.
